Inside the Box

What is this server about?

Inside the box is a puzzle server. You solve puzzles. And you create puzzles. And you share puzzles with other players.

How do I start?

First, you have to obtain a copy of the free Minetest client (version 0.4.16) from minetest.net.

After installing the correct version for your OS, start the client and open the "Play Online" tab. You can find the Inside the box server in the server list, or you can use the text search bar to quickly find it.

Then, choose a username and password, and log into the server. Once you are logged in, it will take a small bit of time to start the game.

I'm here, now what?

When you first log into the game, you will find yourself in The Tutorial , which is a series of 8 puzzle boxes (levels) that you will need to finish.

Make sure you read the server /rules and adhere to the server policies while on the server.

Once you finish the tutorial, you are granted permission to create puzzles yourself, as well as join other players in the lobby, where you can find more challenging puzzles. From here, you can share your creations and play the creations that others have made for you!

Help! I'm stuck!

Check out the Videos we've made to help you through the tutorial levels!

You can also ask for a hint. Since the server is connected to an IRC server (#insidethebox on irc.freenode.net), there are always people online that will be able to help you!
